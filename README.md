The project is a realtime multi-room web chat application built with MERN stack and Websockets
The main features of the chat application:
1. Create your own chat rooms
2. Join someone else's chat room
3. View your joined rooms and compelete chat history
4. Update your user profile

Additional features :
1. Authentication implemented using JWT tokens stored in cookies
2. Passwords hashed and stored in DB instead of storing as plain text
3. Fast and smooth communication system built with socket io
4. Responsive and smooth UI
5. Complete chat history loaded quickly upon joining a room

Upload profile picture is yet to be implemented
